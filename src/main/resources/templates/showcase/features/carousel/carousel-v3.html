Responsive Carousel

OWL Carousel is touch enabled jQuery plugin that lets you create beautiful responsive carousel sliders. You can find more detailed information from the official plugin website http://owlgraphic.com/owlcarousel

Classic style and auto play

You should just add an additional class .owl-carousel-style to make classic style. For auto play you may use .owl-slider class.
